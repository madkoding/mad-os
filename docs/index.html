Capabilities

Key Features

Everything you need in a lightweight and powerful operating system

OpenCode Integration

Integrated AI assistant for intelligent system orchestration and advanced assistance.

Optimized for Low RAM

Specifically designed for systems with 1.9GB RAM (Intel Atom) with ZRAM and EarlyOOM.

Adaptive Desktop

Sway or Hyprland depending on your hardware. With 3D acceleration: Hyprland. Without: Sway. Both with Nord theme and ~300MB RAM.

Built-in WiFi Setup

Connect to your WiFi network directly from the installer. Automatic network scanning with WPA2 support.

6 Languages

Installer available in English, Spanish, French, German, Chinese and Japanese with automatic keyboard layout.

Ready to Develop

Node.js, npm, Git, and VS Code pre-installed to start coding immediately.

Persistent USB

Flash the ISO to a USB and use it directly without installing. Free space is auto-configured as persistent storage.

GTK Installer

GTK graphical installer with Nord theme, intuitive navigation, real-time progress and animated boot splash.

Secure by Default

Locked root, dual UEFI bootloader (NVRAM + fallback), secure passwords and per-user OpenCode isolation.

GPU Compute (CUDA / ROCm)

Pre-installed NVIDIA CUDA and AMD ROCm drivers. Auto-detection at boot with automatic compute activation for AI and parallel computing.
