Capacidades

Características Destacadas

Todo lo que necesitas en un sistema operativo ligero y potente

OpenCode Integration

Asistente de IA integrado para orquestación inteligente del sistema y asistencia avanzada.

Optimizado para Poca RAM

Diseñado específicamente para sistemas con 1.9GB RAM (Intel Atom) con ZRAM y EarlyOOM.

Escritorio Adaptable

Sway o Hyprland según tu hardware. Si hay aceleración 3D usa Hyprland, si no, Sway. Ambos con tema Nord y ~300MB de RAM.

Configuración WiFi Integrada

Conecta a tu red WiFi directamente desde el instalador. Escaneo automático de redes con soporte WPA2.

6 Idiomas

Instalador disponible en inglés, español, francés, alemán, chino y japonés con layout de teclado automático.

Listo para Desarrollar

Node.js, npm, Git y VS Code pre-instalados para comenzar a programar inmediatamente.

USB Persistente

Graba la ISO en un USB y úsala directo sin instalar. El espacio libre se autoconfigura como almacenamiento persistente.

Instalador GTK

Instalador gráfico GTK con tema Nord, navegación intuitiva, progreso en tiempo real y boot splash animado.

Seguro por Defecto

Root bloqueado, bootloader UEFI dual (NVRAM + fallback), contraseñas seguras y OpenCode aislado por usuario.

GPU Compute (CUDA / ROCm)

Drivers NVIDIA CUDA y AMD ROCm pre-instalados. Auto-detección al arrancar y activación automática de compute para IA y cálculo paralelo.
